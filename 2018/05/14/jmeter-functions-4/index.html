Jmeter函数-Calculation篇

发表于 2018-05-14   |  阅读量 次 | 分类于 自动化测试 ， Jmeter

本章节函数类型为Calculation

函数：__counter

  • 作用： 生成递增数
  • 参数:
Attribute Description Required
First argument TRUE if you wish each simulated user’s counter to be kept independent and separate from the other users. FALSE for a global counter. Yes
Second argument A reference name for reusing the value created by this function. Stored values are of the form ${refName}. This allows you to keep one counter and refer to its value in multiple places. No

必选参数原文解释：

TRUE if you wish each simulated user’s counter to be kept independent and separate from the other users. FALSE for a global counter.

可以简单理解选TRUE针对用户递增数；FLASE不限用户全局递增

  • 场景： 线程数：2 循环数：2
    1. 运行次数${__counter(FALSE)} return 1 2 3 4 ==>全局递增
    2. 运行次数${__counter(TRUE)} return 1 2 1 2 ==>某线程=某用户
      可以拼接字符串生成log或查看结果树调试，例如：第${__counter(FALSE)}次测试、第${__counter(FALSE)} 次搜索

函数：__intSum

  • 作用： 用于计算两个或更多个整数值的总和。(引用名称是可选的，但不能是有效的整数。)
  • 参数：
Attribute Description Required
First argument The first int value. Yes
Second argument The second int value. Yes
nth argument The nth int value. No
last argument A reference name for reusing the value computed by this function. If specified, the reference name must contain at least one non-numeric character otherwise it will be treated as another int value to be added. No
  • 例子：
    1. ${__intSum(2,5,MYVAR)} 将返回7（2 + 5）并将结果存储在MYVAR变量中。 所以$ {MYVAR}将等于7。
    2. ${__intSum(2,5,7)} 将返回14（2 + 5 + 7）并将结果存储在MYVAR变量中。
    3. ${__intSum(1,2,5,${MYVAR})} 如果MYVAR值等于8，则返回16，1 + 2 + 5 + $ {MYVAR}）

– ${MYVAR} – 不要选编码 注意jmeter的执行顺序

函数：__longSum

  • 作用： longSum函数可用于计算两个或更多个long值的总和，只要您知道值不会在-2147483648到2147483647之间，则使用此值而不是__intSum。
  • 参数： 与__intSum参数一致
  • 例子： 与__intSum例子一致

函数：__Random

  • 作用： 随机函数返回一个介于给定最小值和最大值之间的随机数。
  • 参数：
Attribute Description Required
Minimum value A number Yes
Maximum value A bigger number Yes
Variable Name A reference name for reusing the value computed by this function. No
  • 例子：
    1. ${__Random(0,10)} 将返回0到10之间的随机数
    2. ${__Random(0,10, MYVAR) 将返回0到10之间的随机数并将其存储在MYVAR中。 ${MYVAR}将包含随机数

函数：__RandomDate

  • 作用：RandomDate函数返回位于给定的开始日期和结束日期值之间的随机日期。
  • 参数：
Attribute Description Required
Time format Format string for DateTimeFormatter (default yyyy-MM-dd) No
Start date The start date, the default is now No
End date The end date Yes
Locale to use for format The string format of a locale. The language code must be lowercase. The country code must be uppercase. The separator must be an underscore, e.g. en_EN. See http://www.oracle.com/technetwork/java/javase/javase7locales-334809.html. If omitted, by default the function uses the Apache JMeter locale one. No
Name of variable The name of the variable to set. No
  • 例子：
    1. ${__RandomDate(,,2050-07-08,,)} 将从现在到2050-07-08之间返回一个随机日期。 例如2039-06-21
    2. ${__RandomDate(dd MM yyyy,,08 07 2050,,)} 将返回一个随机的日期，像04 03 2034这样的自定义格式

函数：__RandomString

  • 作用： RandomString函数使用chars中的字符返回一个随机的字符串长度。
  • 参数：
Attribute Description Required
Length A number length of generated String Yes
Characters to use Chars used to generate String No
Variable Name A reference name for reusing the value computed by this function. No
  • 例子：
    1. ${__RandomString(5)} 将返回一个随机的5个字符串，可以读取或不可读取
    2. ${__RandomString(10,abcdefg)} 将返回从abcdefg集合中选取的10个字符的随机字符串，例如cdbgdbeebd
    3. ${__RandomString(6,a12zeczclk, MYVAR)} 将返回从a12zeczclk集中选取的6个字符的随机字符串，并将结果存储在MYVAR中，MYVAR将包含像2z22ak或z11kce这样的字符串，

函数：__RandomFromMultipleVars

  • 作用： RandomFromMultipleVars函数根据Source Variables提供的变量值返回一个随机值。这些变量可以是简单的或多值的，因为它们可以由以下提取器生成：
    • Regular Expression Extractor
    • CSS/JQuery Extractor
    • JSON Extractor
    • XPath Assertion

Multi-value vars are the ones that are extracted when you set -1 for Match Numbers. This leads to creation of match number variable called varName_matchNr and for each value to the creation of variable varName_n where n = 1, 2, 3 etc.

多值增值是当您为匹配数设置-1时提取的值。 这导致创建名为varName_matchNr的匹配号变量，并为每个值创建变量varName_n，其中n = 1,2,3等。

  • 参数：
Attribute Description Required
Source Variables Variable names separated by竖 that contain the values that will be used as input for random computation Yes
Variable Name A reference name for reusing the value computed by this function. No
  • 例子：
    1. ${__RandomFromMultipleVars(val)}将根据变量值的内容返回一个随机字符串，并考虑它们是否为多值
    2. ${__RandomFromMultipleVars(val1|val2)}将根据变量val1和val2的内容返回一个随机字符串，同时考虑到它们是否为多值
    3. ${__RandomFromMultipleVars(val1|val2, MYVAR)} 会根据变量val1和val2的内容返回一个随机字符串，并考虑它们是否为多值，并将结果存储在MYVAR中

函数：__UUID

  • 参数： 无
  • 例子：
    ${UUID()} 将返回具有以下格式的UUID：c69e0dd1-ac6b-4f2b-8d59-5d4e8743eecd
