Jmeter函数-Properties篇

发表于 2018-05-15   |  阅读量 次 | 分类于 自动化测试 ， Jmeter

本章节函数类型为Properties

函数 __isPropDefined

  • 作用： 如果属性存在，此函数返回true，否则返回false。
  • 参数：
Attribute Description Required
Property Name The Property Name to be used to check if defined Yes
  • 例子：
    ${__isPropDefined(START.HMS)} will return TRUE

    函数 __property

  • 作用： 返回JMeter属性的值。 如果找不到属性值，且没有提供默认值，则返回属性名称。 当提供默认值时，不需要提供函数名称 - 参数可以设置为null，并且将被忽略。 如下：
    • ${__property(user.dir)} 返回user.dir的值
    • ${__property(user.dir,UDIR)} 返回user.dir的值且保存在UDIR
    • ${__property(abcd,ABCD,atod)} 返回abcd属性的值且保存在ABCD (或者 如果”atod”没有定义 )
    • ${__property(abcd,,atod)}返回abcd属性的值 (或者 如果”atod”没有定义) 但不保存
  • 参数：
Attribute Description Required
Property Name The property name to be retrieved. Yes
Variable Name A reference name for reusing the value computed by this function. No
Default Value The default value for the property. No

函数__P

  • 作用： 这是一个简化的属性函数，用于在命令行中定义的属性。 不同于__property功能，没有选项保存在一个变量的值，如果没有缺省值被提供时，它被假定为1。选择的1的值，因为它是有效的常见测试变量，如循环， 线程数，ramp-up等。
    例如：定义属性值jmeter -Jgroup1.threads=7 -Jhostname1=www.realhost.edu
    获取值：
  • ${__P(group1.threads)} 返回group1.threads的值
  • ${__P(group1.loops)} 返回group1.loops的值
  • ${__P(hostname,www.dummy.org)}如果未定义，则返回属性主机名或www.dummy.org的值
    在上面的例子中，第一个函数返回7，第二个函数返回1，最后一个返回www.dummy.org除非这些属性是在别处定义的！
  • 参数：
Attribute Description Required
Property Name The property name to be retrieved. Yes
Default Value The default value for the property. If omitted, the default is set to “1”. No

函数 __setProperty

  • 作用： setProperty函数设置JMeter属性的值。 函数的默认返回值是空字符串，因此可以在函数有效的任何位置使用函数调用。

    • 通过将可选的第三个参数设置为“true”可以返回原始值。
    • 属性对于JMeter是全局的，因此可用于在线程和线程组之间进行通信

  • 参数：

Attribute Description Required
Property Name The property name to be set. Yes
Property Value The value for the property. Yes
True/False Should the original value be returned? No

函数 __isVarDefined

  • 作用： 如果变量存在，函数返回true;否则返回false。
  • 参数：
Attribute Description Required
Variable Name The Variable Name to be used to check if defined Yes
  • 例子：
    ${__isVarDefined(JMeterThread.last_sample_ok)} 返回True
