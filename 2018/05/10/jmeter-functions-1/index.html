Jmeter函数-information篇

发表于 2018-05-10   |  阅读量 次 | 分类于 自动化测试 ， Jmeter

本章节为information函数类型，多用于打印log调试用。

PS.补充官方Parameters和Examples

函数：__threadNum

  • 作用： 获取当前线程（用户）编号
  • 场景案例： 线程1录入user1 input ;线程2录入user2 input;……………
    在sampler中的parameters录入user${__threadNum} input
    适用场景在log中调试用

函数：__samplerName

  • 作用： 返回当前sampler的名称
  • 参数：
Attribute Description Required
Variable Name A reference name - refName - for reusing the value created by this function. Stored values are of the form ${refName}. No
  • 例子： ${__samplerName()}
  • 场景案例： 也是用于log中调试

    函数：__macheineIP

  • 作用： 函数返回本地IP地址
  • 参数：
Attribute Description Required
Variable Name A reference name for reusing the value computed by this function. No
  • 例子： ${__machineIP(####)} 和${__machineIP} 都返回- 本机IP
  • 场景案例： 1、还是log- 2、分布式

函数：__machineName

  • 作用： 函数返回本地机器的名称
  • 参数：
Attribute Description Required
Variable Name A reference name for reusing the value computed by this function. No
  • 例子： ${__machineName()}和${__machineName} 都返回hostname
  • 场景案例： 还是log

函数：__time

  • 作用： 返回各种格式的当前时间
  • 参数：
Attribute Description Required
Format The format to be passed to SimpleDateFormat. The function supports various shorthand aliases, see below. If omitted, the function returns the current time in milliseconds since the epoch. No
Name of variable The name of the variable to set. No

  • 简写格式如下：

  • 例子：

    ${__time(dd/MM/yyyy,)}return 21/01/2018

    ${__time(YMD,)} return20180121

    ${__time()} return 当前时间戳

    还有其他表达方式:

    ${__time(EEE\, dd MMM yyyy)}return Thu,10 May 2018

函数：__timeShift

  • 作用： timeShift函数返回指定格式的日期，并添加指定的秒数，分钟数，小时数，天数或月数
  • 参数：
Attribute Description Required
Format The format to be passed to DateTimeFormatter. See DateTimeFormatter If omitted, the function uses milliseconds since epoch format. No
Date to shift Indicate the date in the format set by the parameter ‘Format’ to shift If omitted, the date is set to now No
value to shift Indicate the specified amount of seconds, minutes, hours or days to shift according to a textual representation of a duration such as PnDTnHnMn.nS. See https://docs.oracle.com/javase/8/docs/api/java/time/Duration.html#parse-java.lang.CharSequence- No
Locale to use for format The string format of a locale. The language code must be lowercase. The country code must be uppercase. The separator must be an underscore. ex: en_EN See http://www.oracle.com/technetwork/java/javase/javase7locales-334809.html If omitted, by default the function use the ApacheJMeter locale one. No
Name of variable The name of the variable to set. No
  • 例子：
    • ${__timeShift(dd/MM/yyyy,21/01/2018,P2D,,)} 返回23/01/2018
    • ${__timeShift(dd MMMM yyyy,21 février 2018,P2D,fr_FR,)} 返回23 février 2018

函数：__log

  • 作用： 在log中打印
  • 参数：
Attribute Description Required
String to be logged A string Yes
Log Level OUT, ERR, DEBUG, INFO (default), WARN or ERROR No
Throwable text If non-empty, creates a Throwable to pass to the logger No
Comment If present, it is displayed in the string. Useful for identifying what is being logged. No

OUT和ERR日志级别名称分别用于将输出指向System.out和System.err。 在这种情况下，输出总是被打印 - 它不依赖于当前的日志设置。

  • 场景实现：
    在前置、后置处理器及线程组中可以打印log，输出在log和view Results tree
  • log和view Results tree的区别
    log中可以持久化，viewResults Tree 运行线程组会覆盖当前信息

函数：__logn

  • 作用： 原意：logn函数记录一条消息，并返回空字符串（和log相似，但只输出在log里）
  • 参数：
Attribute Description Required
String to be logged A string Yes
Log Level OUT, ERR, DEBUG, INFO (default), WARN or ERROR No
Throwable text If non-empty, creates a Throwable to pass to the logger No
  • 例子：
    ${__logn(VAR1=${VAR1},OUT)}将变量的值写入控制台窗口
