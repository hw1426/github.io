Vultr梯子记

发表于 2018-04-16   |  阅读量 次 | 分类于 Linux ， SSR

早年vultr、搬瓦工搭酸酸乳在江湖上如雷贯耳，甚是向往，但一直未碰见2.5刀的廉价vps，最近上班摸鱼发现vultr有活动（新用户赠送25刀），没忍住开了新账户充了10刀，随后发现有Newyork和Miami的2.5刀套餐，虽然不是最理想的机房，但胜在廉价，抱着试一试的态度购买后按照教程搭好梯子，效果意外的不错！此次过程记录如下：

安装BBR

BBR 是 Google 提出的一种新型拥塞控制算法，可以使 Linux 服务器显著地提高吞吐量和减少 TCP 连接的延迟。
BBR解决了两个问题：

  • 再有一定丢包率的网络链路上充分利用带宽。非常适合高延迟，高带宽的网络链路。
  • 降低网络链路上的buffer占用率，从而降低延迟。非常适合慢速接入网络的用户。 此次安装参考了一键安装脚本进行安装：

  1. 使用root用户登陆，运行：

    wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh && chmod +x bbr.sh && ./bbr.sh

  2. 验证是否成功安装最新内核并开启 TCP BBR 运行：

    sysctl net.ipv4.tcp_available_congestion_control

    返回值一般：
    net.ipv4.tcp_available_congestion_control = bbr cubic reno 说明BBR已完成安装。

    输入命令：

    lsmod | grep bbr

    返回值有 tcp_bbr 模块即说明 BBR已启动。

搭梯子

Server

  1. 使用root用户登陆，运行：

    wget –no-check-certificate  https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks.sh

  2. 运行：

    chmod +x shadowsocks.sh

  3. 运行：

    ./shadowsocks.sh 2>&1 | tee shadowsocks.log

    中间会提示你输入你的SS SERVER的账号，和端口。不输入就是默认。跑完命令后会出来你的SS客户端的信息。安装完成后，脚本提示如下：
