Jmeter-Variables篇

发表于 2018-05-15 | 分类于 自动化测试 ， Jmeter

函数类型为Variables及Formatting

函数：__V

  • 作用： V（变量）函数返回计算变量名称表达式的结果。 这可以用来评估嵌套变量引用（目前不支持）。

  • 例子：
    如果有变量A1，A2和N = 1：则如下

    • ${A1} //works OK

    • ${A${N}} //does not work

    • ${__V(A${N})} //work OK

  • 参数：

Attribute Description Required
Variable name The variable to be evaluated. Yes

常用于变量命拼接

  • 场景案例：
    以下为用户自定义变量

    • name1 = tim
    • name2 = tom
    • name3 = jack

    使用${__V(name${__threadNum})}调试

函数：__eval

  • 作用： 返回了一个字符串的表达式的值。可以在存储在变量中的字符串中插入变量和函数引用。 例如，给定以下变量：

    • name = tim
    • column = age
    • table = birthdays

    SQL = select ${column} from ${table} where name = ‘${name}’
    则：${__eval(${SQL})} evaluate as “select age from birthdays where name = ‘tim’”

    这可以与CSV数据集一起使用，例如在数据文件中定义了SQL语句和值。（常用于动态拼接字符串）

  • 参数：
Attribute Description Required
Variable name The variable to be evaluated. Yes

  • 场景案例：
    以下为用户自定义变量

    • name1 = tim
    • name2 = tom
    • name3 = jack

    则：blogname = ${__V(name${__threadNum})}的blog
    tim的blog=${__eval(${blogname})}

函数__evalVar

  • 略

函数 __split

  • 略

函数__dateTimeConvert

  • 略
