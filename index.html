Atom_Error

发表于 2018-05-16

目前使用Atom作为主力编辑器，本周Atom启动报错Cannot load the system dictionary for zh-CN如图: 引起莫名闪退,编辑文件体验相当差，先后尝试更新Atom版本（已更新至1.27.0）及禁用中文插件，都没有疗效。

Google后在GitHub上找的大佬正确解决姿势,如下：

  • 文件-设置-插件-核心插件-spell-check-禁用。
  • File-Settings-Packages-Core Packages-spell-check-disable。

Jmeter相关快捷键

暂时收集一些常用的快捷键，如：Jmeter三连击

快捷键 功能
Ctrl + S 保存
Ctrl + E 清除运行结果
Ctrl + R 运行
Ctrl + T 禁用or启用组件
Ctrl + Shift + C 复制当前组件到下一行
Ctrl + F 搜索
Ctrl + - 收起测试计划
Ctrl + Shift + - 测试计划展开
Ctrl + G 当前节点截屏
Ctrl + Shift + G 当前JMeter窗口截屏
Ctrl + 0 线程组
Ctrl + 1 HTTP请求
Ctrl + 2 正则表达式提取器
Ctrl + 3 响应断言
Ctrl + 4 固定定时器
Ctrl + 5 Test Action
Ctrl + 6 JSR223 PostProcessor
Ctrl + 7 JSR223 PreProcessor
Ctrl + 8 Debug Sampler
Ctrl + 9 查看结果树
