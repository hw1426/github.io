jmeter-函数String篇

本章节函数类型为String

函数：__regexFunction

  • 作用： 正则表达式解析

函数：__escapeOroRegexpChars

  • 作用： 正则表达式中的保留字符转译

函数：__char

  • 作用： unicode编码转unicode字符串
  • 例子: TODO

函数：__unescape

  • 作用： 调用Java反转译，字符串转译编码
  • 例子： TODO

函数：__unescapeHtml

  • 原文介绍：

    Function to unescape a string containing HTML entity escapes to a string containing the actual Unicode characters corresponding to the escapes.

    富HTML转译HTML ==> 引申XXS攻击

  • 例子：A=&lt;script&gt;xxxxxxxx&lt;/script&gt; 转译为<script>xxxxxxxx</script>

    {__unescapeHtml(A)} will return <script>xxxxxxxx</script>

函数：__escapeHtml

  • 作用 和__unescapeHtml 相反

函数：__escapeXml

  • 作用: 与__escapeHtml 一样，作用于XML

  • 原文：

    Function which escapes the characters in a String using XML 1.0 entities.

  • 例子： ${__escapeXml("bread" & 'butter')}
    returns: &quot;bread&quot; &amp; &apos;butter&apos;

函数：__urldecode

  • 作用： 转译过的字符串，反转译
  • 例子1：
    ${__urldecode(Word+%22school%22+is+%22%C3%A9cole%22+in+french)}
    returns: Word "school" is "école" in french
  • 例子2：
    ${__urldecode(%E5%85%A8%E5%9B%BD%E4%BA%BA%E6%B0%91%E5%96%9C%E8%BF%8E%E6%B2%B9%E4%BB%B7%E4%B8%8A%E6%B6%A8)}
    returns: 全国人民喜迎油价上涨

函数：__urlencode

  • 作用： 与__urldecode相反
  • 例子：
    ${__urlencode(全国人民喜迎油价上涨)}
    returns: %E5%85%A8%E5%9B%BD%E4%BA%BA%E6%B0%91%E5%96%9C%E8%BF%8E%E6%B2%B9%E4%BB%B7%E4%B8%8A%E6%B6%A8

函数：__TestPlanName

  • 作用： 返回当前测试计划的名称
  • 场景案例：
    jmeter批处理脚本一次执行多个testplan,可以用此函数把每个plan的名字打印在log。 （强行找场景，也许有用。一般用于调试）

jmeter函数-input篇

发表于 2018-05-11

本章节函数类型为input，此类函数可读取TXT、CSV、XML、DAT格式文件，常用于测试数据 （文件常放在/bin目录下？）

函数：__StringFromFile

  • 作用： Read a line from a file 文件中读取一行数据

    1. 数据文件支持多格式文件（.dat , .txt , .csv）；

    2. 数据文件和脚本放在 同一文件夹下（没实现，待验证），绝对路径已验证。

      PS.更正相对路径已验证，数据文件应放在jmeter/bin路径下

  • 例子： 读取D盘下test.txt文件（一行）
    函数表达为：${__StringFromFile(D:\test.txt,,,)}

    注意函数中有三个可选参数（Variable Name、Start sequence number、End sequence number），不选参数也要加，

    PS.乙醇没有讲好坑啊！ 给函数助手点赞，可以对不熟悉的函数调试

函数：__FileToString

  • 作用： Read an entire file 读取数据文件全文（此函数可选参数2个）

  • 例子： 读取读取D盘下test.txt文件（全文）

    函数表达为：${__FileToString(D:\test.txt,,)}
    可选参数（File encoding if not the platform default、Variable Name）

函数：__CSVRead

  • 作用： Read from CSV delimited file 读CSV文件,传入参数有两个必选参数（File Name、Column number）
    调用函数需要CSV文件名（及路径）和CSV文件列数 第一列起始为0

    Column number原文解释：The column number in the file. 0 = first column, 1 = second etc. “next” - go to next line of file.

  • 例子：

    读取不方便建议用配置原件中的CSV Date Set Config代替使用

    官方例子说明__CSVRead也可以读取txt文件，这和__StringFromFile有什么区别？

函数：__XPath

  • 作用： Use an XPath expression to read from a file

    读取XML文件并匹配XPath,调用函数，返回下个匹配项；读取在文件末，它将循环至开始；没有匹配节点则返回空字符串并ERR信息记录在Jmeter日志文件内。

Jmeter函数-information篇

发表于 2018-05-10

本章节为information函数类型，多用于打印log调试用。

函数：__threadNum

  • 作用： 获取当前线程（用户）编号
  • 场景案例： 线程1录入user1 input ;线程2录入user2 input;……………
    在sampler中的parameters录入user${__threadNum} input
    适用场景在log中调试用

函数：__samplerName

  • 作用： 返回当前sampler的名称
  • 例子： ${__samplerName()}
  • 场景案例： 也是用于log中调试

函数：__macheineIP

  • 作用： 函数返回本地IP地址
  • 例子： ${__machineIP(#### )} 和${__machineIP} 都返回- 本机IP
  • 场景案例： 1、还是log- 2、分布式

函数：__machineName

  • 作用： 函数返回本地机器的名称
  • 例子： ${__machineName()}和${__machineName} 都返回hostname
  • 场景案例： 还是log

函数：__time

  • 作用： 返回各种格式的当前时间

  • 格式如下：

  • 例子：

    ${__time(dd/MM/yyyy,)}return 21/01/2018

    ${__time(YMD,)} return20180121

    ${__time()} return 当前时间戳

    还有其他表达方式:

    ${__time(EEE\, dd MMM yyyy)}return Thu,10 May 2018
